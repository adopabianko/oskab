## Installation

clone project

``
git clone https://github.com/adopabianko/oskab.git
``

copy file .env

``
cp -R .env.example .env
``

run composer install

``
composer install
``

run laravel sail

``
./vendor/bin/sail up
``

access http://localhost:9595 via browser
